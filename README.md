# Objetivo <br>

O projeto consiste em uma 'WebApi' desenvolvida para servir de insumo a uma das fases do processo seletivo da empresa B2W. <br>

# Ferramentas e frameworks utilizados <br>
- Java 11<br>
- IDE STS<br>
- Spring Boot<br>
- MongoDB<br>
- Swagger<br>
- WebFlux<br>
- JUnit<br>
- REST Assured<br>

# Executando o projeto no STS <br>

Clicar com o botão direito do mouse sobre o arquivo "DesafioB2wApplication.java' e escolher as opções "Run As > Spring Boot App" <br>

# Executando os testes de Banco e Api <br>

Clicar com o botão direito do mouse sobre o arquivo "DesafioB2wApplicationTests.java" e escolher as opções "Run As > JUnit Test" <br>

# Documentação <br>

Breve manual da API encontra-se na raiz do projeto, em um arquivo chamado "StarWars-Api-Manual.pdf" <br>

# Projeto Postman <br>

No diretório raiz do projeto há um arquivo chamado "StarWars.postman_collection.json"<br>
